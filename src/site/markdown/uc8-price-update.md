# ΠΧ8. Ανανέωση Τιμών Μετοχών

**Πρωτεύων Actor**: Σύστημα Χρηματιστηρίου 
**Ενδιαφερόμενοι**
**Επενδυτής**: Θέλει να ενημερωθεί για τις τελευταίες τιμές μετοχών
**Μεσίτης**: Θέλει να ενημερωθεί για τις τελευταίες τιμές μετοχών
**Προϋποθέσεις**: Ο Επενδυτής και ο Μεσίτης να έχουν εκτελέσει με επιτυχία την περίπτωση χρήσης Δημιουργία Προφίλ.

## Βασική Ροή

### Α) Ανανέωση Τιμών Μετοχών

1. Στο τέλος της μέρας ενεργοποιείται η Εκτέλεση αυτόματων Εντολών
2. Το Σύστημα Χρηματιστηρίου αποστέλει τις ανανεωμένες τιμές μετοχών.
3. Το Σύστημα μας ενημερώνει τις τιμές μετοχών.
